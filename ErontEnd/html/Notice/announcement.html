简介
Vue前端框架 ： Vue前端框架是用于构建用户界面的渐进式框架，采用自下而上增量式开发设计。

Element ： Element是一套以 Vue2.0为基础桌面组件库，供开发者、设计师和产品管理人员使用。
MySQL ： MySQL是一个关系型数据库，与其他流行的数据库相比，它可能没有最好的功能和性能，但MySQL是免费和开源的，并能完成基本功能。

Java ： Java是一种面向对象的语言，它在继承了C++的优势的同时，也抛弃了C++中那些晦涩难懂的多继承、指针等概念。
Vs Code ： Vs Code的全名是 Visual Studio Code，这是微软公司在 Build开发者会议上正式发布的一款跨平台源代码编辑器，用于编写面向当代网络和云计算的应用。

IDEA ： IDEA的全名是IntelliJ IDEA，是一种整合的Java程序语言开发环境。IntelliJ被认为是最优秀的Java开发工具，特别是智能代码助手，自动代码提示，重构，JavaEE支持，各种版本工具，Junit，CVS集成，代码分析，创新GUI设计。
针对数据库进行优化：受于技术限制，在数据库方面，查询，更改，数据多了会出现混乱的效果，在日后急需完善

前端展示界面 ：由于采用的是vue-element的开发方式，在标签使用上还不完善，需要在后期进行大幅度的更改

后端命令优化 ：使用框架较少，在不断的学习中将采用Lombok、Spring、等多种的框架，精简代码量，达到占用小，系统更加灵活的效果
